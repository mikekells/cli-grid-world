# CLI Grid World

## Overview

CLI Grid World is a personal Java learning project focused on building a structured game architecture from first principles.

This project began as a command-line based world simulation designed to strengthen my understanding of:

* Object-Oriented Programming (OOP)
* Game loops
* State management
* Separation of concerns
* Clean project structure
* Git and professional development workflows

The long-term goal is to evolve these foundations toward graphical 2D game development.

---

## Purpose

This repository intentionally documents **learning, struggle, iteration, and growth**.

Rather than following tutorials, features are rebuilt repeatedly from memory and experimentation to develop genuine understanding.

Mistakes, refactors, and redesigns are considered part of the learning process.

---

## Current Features

* Grid-based world rendered in CLI
* Player movement system
* World boundary validation
* Game state management
* Modular class structure:

  * `GameState`
  * `World`
  * `Renderer`
  * `Movement`
  * `Position`

---

## Learning Goals

This project focuses on developing instinctive understanding of:

* Designing systems before implementation
* Data vs behaviour separation
* Game architecture fundamentals
* Debugging logic independently
* Version control using Git & GitHub
* Incremental feature development via branches
